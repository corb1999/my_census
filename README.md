# my_census
Use tidycensus to get quick measures and visuals/maps for personal research

resources to explore and try implementing:
- https://walker-data.com/tidycensus/
- https://walker-data.com/census-r/index.html
- https://rpubs.com/dieghernan/beautifulmaps_I (learn geo-binning techniques)
- http://davidgohel.github.io/ggiraph/index.html (learn interactive maps/plots)
- https://docs.ropensci.org/nasapower/ (get rainfall data)
- https://docs.ropensci.org/osmdata/ (streets and buildings, open street data)
